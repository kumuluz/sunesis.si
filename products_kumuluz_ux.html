Sophisticated framework for enterprise-grade web and mobile user interface with superb experience

KumuluzUX Web/Mobile Framework provides a rich set of advanced HTML5 and JSF components for rapid development of enterprise grade web and mobile user interfaces with superf user experience. The competitive advantage of the KumuluzUX are sophisticated composite components for rapid development of web forms, human tasks, business process interaction, reports and other common interfaces in enterprise information systems. KumuluzUX includes custom components for enterprise-grade user interfaces from banking, financial, insurance and power distribution domains. With KumuluzUX, such interfaces and/or forms and reports can be developed with single line of code.

KumuluzUX provides smart, configurable components, which can generate layouts and render the content directly from data source, ranging from APIs, services, SOAP, REST, OData to data sources such as JDBC, ODBC and JPA.

KumuluzUX provides tight BPM integration with a rich set of component for rapid development of Process Applications using IBM BPM, Oracle SOA/BPM Suite and/or JBoss jBPM. Furthermore, it provides specific components for state management and reporting.

Features:
• Rich set of advanced HTML5 and JSF components for rapid development of enterprise grade web and mobile user interfaces
• Full support for BPM and human task integration with integration with IBM BPM, Oracle SOA/BPM Suite and JBoss jBPM
• Business session state management
• Easy integration of different applications into common UI
• Reporting
• Domain specific components for banking, finance, insurance and power distribution
• Native and
• Superb user experience

About company

Sunesis name was not chosen randomly, but communicates the core values represented by the company. The Greeks had three great words describing three great qualities of the mind. If a man possessed these three qualities he had a mind equipped. The first of these words is Sophia - wisdom of ultimate things. The second of the three great words is Phronesis - prudence. The third of the great Greek words of the mind is Sunesis. Sunesis literally means uniting, bringing together, integrating. It would be true to say that Sunesis is that faculty of putting two and two together. In its essence Sunesis is the power to asses, distinguish, criticize and evaluate, understand and to form judgments.
